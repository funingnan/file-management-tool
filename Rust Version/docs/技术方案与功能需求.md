# Rust + egui 版本技术方案

## 技术栈
- GUI: eframe + egui 0.31
- 数据库: rusqlite (bundled)
- 文件遍历: walkdir
- 文件对话框: rfd
- 序列化: serde + serde_json

## 体积控制
```toml
[profile.release]
opt-level = "z"
lto = true
codegen-units = 1
panic = "abort"
strip = true
```
预期: 5~8MB，零依赖

## 编译
```bash
cargo build --release
```
输出: target/release/pdf-knowledge-base.exe
